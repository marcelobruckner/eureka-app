# Scripts de banco

Registre aqui toda alteracao estrutural de banco de dados.

Padrao de nome sugerido:

- `001_schema_inicial.sql`
- `002_adiciona_indice_tarefas_status.sql`

No H2/local, alteracoes diretas sao permitidas para validacao rapida, mas devem ser refletidas nestes scripts. Para PostgreSQL, a estrutura deve ser criada a partir dos scripts versionados.
